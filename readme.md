## 2. 介绍

- 项目参考 [米哈游官网-国内](https://www.mihoyo.com) , 进行粗略的模仿
- 项目图片视频等资源来自米哈游官网, 仅编程学习用
- 因为是按照2550X1440分辨率的显示屏来开发的. 未考虑适配问题, 所以可能会出现图片大小位置出错的情况. 使用同分辨率的屏幕应该正常.
- 能力有限, 无法做到米家那般丝滑
- 项目应该还存在许多问题(ノへ￣、)
- 个人学习自用
